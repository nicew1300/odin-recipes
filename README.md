# odin-recipes
This was my very first project in The Odin Project. I created multiple pages about my favorite dishes using basic HTML.
I had to set up relative links between pages, add images, and create both ordered and unordered lists. I also learned how to find free-to-use images safely, using Unsplash.com. The project was pretty easy since I already knew the basics of HTML, but it helped me get used to structuring multiple pages and linking them together.
In this project, the newly used technologies were multi-page HTML structure, anchor links, image elements, and lists.
